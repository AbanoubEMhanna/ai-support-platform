# ai-support-platform

AI customer support SaaS (portfolio project) showcasing:

- Multi-tenant backend (NestJS)
- Async processing (RabbitMQ)
- Vector search (Postgres + pgvector)
- RAG chat (OpenAI)
- Web dashboard (Next.js)

## Monorepo structure

```txt
apps/api       # NestJS API
apps/web       # Next.js web app
packages/shared
docs
docker
```

## Getting started

```bash
pnpm -v
pnpm install
```
